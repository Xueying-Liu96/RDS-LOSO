# R Code for *New Perspectives on Respondent-Driven Sampling with Application to Sex-Trafficked Population in Senegal*

This repository contains the R code for the data analysis in the article **“New Perspectives on Respondent-Driven Sampling with Application to Sex-Trafficked Population in Senegal,”** published in the *Journal of Statistical Theory and Practice*.

A 20-sample de-identified dataset is included for demonstration purposes only. The original study data are not included in this repository due to confidentiality and privacy considerations.

Readers may refer to the following report for more details on the original data collection:

Okech, D., Kelly, J., Mbaye Soukeye, M., Cronberg, A., Yi, H., Cody, A. M., ... & Diagne Barre, A. (2022). *Sex Trafficking in the Gold Mining Areas of Kédougou, Senegal: A Mixed-Methods Study Estimating Baseline Prevalence and Identifying Perceived Gaps in Prevention, Prosecution, and Protection Response*. Center on Human Trafficking Research & Outreach, University of Georgia. https://doi.org/10.71927/uga.26739
